# MyConfig

我常用的VSCode配置，使用XMake重新构建。

本配置主要针对C/C++的单文件Demo级代码，用于学习和算法竞赛（Competitive Programming）

## 依赖

- XMake
- gcc@14
- XCode 15.3

## 使用方法

如果想生成`compile_commands.json` 请使用`VSCode`的`Task`功能，在`task.json`中，我编写了快速生成`compile_commands.json`的任务脚本。
